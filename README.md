
## Yükleme

```bash 
  cd contactapp
```

```bash 
  cp .env.example .env
```
  
```bash 
  docker-compose up -d
```
  
```bash 
  composer install
```

```bash 
  yarn && yarn build
```
  
```bash 
  php artisan migrate --seed
```
  
```bash 
  php artisan passport:client --personal -n
```
  
```bash 
  php artisan key:generate
``` 

```bash 
  php artisan serve
```

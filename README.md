
## Yükleme

```bash 
  cd contactapp
```

```bash 
  cp .env.example .env
```
  
```bash 
  docker-compose up -d
```
  
```bash 
  composer install
```
  
```bash 
  php artisan:migrate --seed
```
  
```bash 
  php passport:client --personal -n
```
  
```bash 
  yarn build
```
